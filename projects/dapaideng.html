Vibe Coding Project · 01

大排灯打卡

这是我的第一个上线的 Vibe Coding 小程序：把每天照灯这件小事，变成一个可以坚持的仪式。

The starting point

照灯不是难事，难的是每天记得做。

一次照灯会涉及不同部位、光照模式和时长。这个小程序把方案、每日任务、连续记录和设备信息放进同一条路径，让照灯这件事更容易开始，也更容易坚持。

WeChat Mini Program Vibe Coding Personal Project

Inside the app

从安排到复盘，一个完整的小循环。

01

今天照什么，一眼看清

把不同部位、光照模式和时长整理为当天任务，并支持一键完成全部打卡。

02

把计划变成可以执行的七天方案

脸部、背部和头部任务按天编排，每一天需要做什么、做多久，都能直接看到。

03

让坚持变得看得见

连续天数、累计记录、使用成本和每日完成情况集中在同一个日历视图里。

04

设备信息也放在同一个地方

记录当前绑定设备，并把不同版本的功能、功率和照射面积放在一起比较。

First shipped experiment

第一次把 Vibe Coding 产物真正放到用户手里。

它从一个非常具体的生活需求出发，经过页面、流程和数据结构的反复调整，最后成为一个可以被使用的小程序。比起功能有多少，更重要的是把想法完成并发布。
